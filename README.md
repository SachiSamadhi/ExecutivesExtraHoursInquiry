Executives Extra Hours Inquiry System

The Executives Extra Hours Inquiry System is an internal enterprise application designed to track, analyze, and review extra working hours performed by executive-level employees. The system provides an efficient and structured way for HR and management teams to inquire into attendance records, overtime details, and approval-related information within a selected period.
